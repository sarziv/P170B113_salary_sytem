# P170B114_salary_sytem
### Lab Ataskaitos
##### Lab ataskaitos folder

#### Projektuojama informacinė sistema
Algų skaičiavimo sistema. Pagrindiniai sistemos vartotojai yra Darbuotojai, kurie saugo apdirbtą žemės dalį kiekvieną darbo dieną. Darbuotojų informacija yra apdirbama buhalterijos ir skaičiuojami atitinkamai uždirbti atlyginimai. Sistema prižiūri administracija.

#### Naudotojų dalies valdymas
Funkcijos priklauso visiems vartotojams, iš kurių susideda prisijungimas/registracija/atsijungimas, visiems naudotojams priklauso pranešimų siuntimo sistema, jos pagrindinės funkcijos pranešimų siuntimas, pranešimų ištrynimas ir vartotojų pranešimų blokavimas, visos pranešimų kopijos bus gaunamos automatiškai nurodytu El.paštu.

####  Administracijos dalies valdymas
Administracijos dalis prižiūri prie sistemos dirbančius darbuotojus, kurie yra buhalterijos darbuotojai ir kiti administratoriai.
Administracijos funkcijos naujų administratorių ir buhalterijos darbuotojų sukūrimas, redagavimas, ištrynimas ir blokavimas.

#### Buhalterinės dalies valdymas
Buhalterijos dalis yra atsakinga už darbuotojų atlyginimų skaičiavimą ir darbuotojų premijų priskyrimą, taip pat atostogų pagal darbuotojų prašymus priskyrimą ir darbuotojų nusižengimų informacijos saugojimą.

#### Darbuotojų dalies valdymas
Darbuotojai atsakingi už informacijos pildymą apie apdirbtą žemę, kuriai priklauso funkcijos pildymas, redagavimas ir ištrynimas. Papildomos funkcijos apie darbuotojų darbo grafiką informaciją, uždirbtų atlyginimų peržiūrėjimą ir atostogų prašymą.
